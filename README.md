# Powers

An interactive geopolitical power-mapping website. Explore a layered world map, regional deep-dives, and country profiles.

## Stack

- Next.js (App Router)
- D3.js + TopoJSON
- Tailwind CSS
- Markdown for analysis content

## Getting Started

```bash
npm install
npm run dev
```

## Data

All geopolitical data lives in `data/` as JSON files. Regional analysis is written in Markdown under `content/regions/`.

See `docs/plans/2025-02-11-powers-site-design.md` for the full design document.
